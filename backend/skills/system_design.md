系统设计
考察：

基础理论：CAP 定理与取舍、BASE 理论、一致性哈希原理、负载均衡策略（轮询/加权/最少连接）
高并发处理：限流算法（令牌桶/漏桶/滑动窗口）、服务降级策略、熔断器模式、缓存穿透/击穿/雪崩解决方案
数据库设计：分库分表方案（水平拆分/垂直拆分）、读写分离架构、数据迁移策略、ShardingSphere 使用
消息系统：消息投递语义、Kafka 架构（Producer/Partition/Consumer/Offset）、消息积压处理方案
搜索系统：倒排索引原理、TF-IDF 算法、BM25 排序、Elasticsearch 分片与副本机制
分布式基础：分布式 ID 生成、分布式锁实现、分布式事务（2PC/TCC/Saga）、分布式 Session 方案
经典系统设计：短链接系统、秒杀系统、Feed 流系统、即时通讯系统、网盘系统、地理位置服务
